# ☁️ Yelling at the Cloud

A status page for cloud services I care about, presented by everyone's favorite elderly cartoon man.

**https://yellingatthe.cloud**

## What is this?

When cloud services are down, Abe Simpson yells at the cloud. When everything's fine, he sleeps peacefully.

That's it. That's the whole thing.

## Services Monitored

- Anthropic
- AWS
- Cloudflare
- GitHub
- OpenAI

These are just the services *I* use. This is not a comprehensive status aggregator.

## Should I use this?

Probably not for anything serious. Check the actual status pages if you need reliable information:

- https://status.anthropic.com
- https://health.aws.amazon.com
- https://www.cloudflarestatus.com
- https://www.githubstatus.com
- https://status.openai.com

## Tech

Single HTML file. No build step. No framework. No npm. Just vibes.

Open `index.html` in a browser or deploy to GitHub Pages.
